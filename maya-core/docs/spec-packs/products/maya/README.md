# Maya Spec Pack v0.1

Dieses Spec Pack ist eine reine Dokumentations- und Ordnungsbasis für Maya im Ordner `maya-core`.

Es trennt strikt zwischen:
- aktuellem Ist-Zustand
- Zielbild und Spezifikationsregeln
- offenen oder unklaren Punkten

Das Pack ist kein Freibrief für Produktumbau, Runtime-Eingriffe oder implizite Architekturentscheidungen.

## Lesereihenfolge

1. `STATUS.md`
2. `CANON.md`
3. `BLUEPRINT.md`
4. `CONTRACT.md`
5. `EMBODIMENTS.md`
6. `CONTINUITY.md`

## Änderungsregel

Neue Ideen, Umbauten oder Architekturvorschläge gehen zuerst als Proposal außerhalb dieses Packs.

Erst danach dürfen belastbare und geklärte Punkte in dieses Pack übernommen werden.
